硬件系统环境需求

一台具有独立外网IP的服务器，可以租用阿里云或者腾讯云的，比较稳定，配置要求如下：

  • CPU：2核
  • 内存：4GB
  • 网络：1MB/s（支持10人），5MB/s支持（32人），建议选择2MB/s及以上的
  • 硬盘：10GB以上
  • 系统：Linux（centos 7.5）或Windows server 2018以上版本

cs1.6服务端资源下载

  1. Linux操作系统，请点击这里下载：rehlds_linux_server
  2. Windows操作系统，请点击这里下载：rehlds_windows_server
  3. 娱乐服插件包，同时支持Linux和Windows系统，请下载：2023娱乐服插件包

娱乐服插件包说明

  1. 内置czip模块，支持对qqwry纯真IP库信息的查询读取
  2. 内置czfun插件，支持观察者HUD信息，支持击杀回血回甲，支持游戏类型修改，支持加入游戏玩家信息提醒，支持玩家击杀信息提醒等，均可在amxx.cfg配置
  3. 在PK类型地图中，回合时间倒计时结束后，强制结束本回合，并开始新回合
  4. 自带zbot，可自动生成waypoint路点，每张地图第一次加载时，会自动分析并生成路点（可能会卡一小会儿）
  5. 其他默认配置，可以在此插件包中的cfg或ini文件重新配置

  • SafeNameAndChat（v1.1）：snac，可以替换玩家昵称和聊天消息中的特殊字符（防止恶意攻击或服务器崩溃）。此外，它还会删除开头和结尾的不可打印字符和空格。
  • VoiceTranscoder（v2017RC5）：vtc，允许Steam/Non Steam客户端语音说话通信，有了此插件正版和盗版玩家之间才能听到对方按K键说话。
  • WHBlocker（v1.5.697）：whblocker，阻止WallHack透视作弊器，如OpenGL32和audible ES等类型的透视作弊器。

娱乐服搭建方法

在Windows搭建

  1. 关闭防火墙（或者配置安全策略允许所有入站出站）
  2. 远程登录windows服务器，下载windows版本的rehlds服务端，并解压
  3. 下载娱乐服插件包，解压出cstrike文件夹和hlds.bat文件，并覆盖rehlds服务端的同名文件夹和文件
  4. 运行hlds.bat即可完成服务器搭建

其他，关于zbot机器人的管理，可以参考文档底部的参考文献

在Linux搭建

  1. 关闭防火墙
  2. 下载Linux版本的rehlds服务端，并解压上传到Linux服务器的用户目录的rehlds_linux_server目录
  3. 安装32位系统兼容组件，链接steamclient.so文件到用户目录的~/.steam/sdk32/目录
  4. 下载娱乐服插件包，解压出cstrike，并覆盖rehlds服务端的cstrike文件夹
  5. 在Linux安装screen软件（让cs服务端在后台运行，这里不对使用说明进行细说）
