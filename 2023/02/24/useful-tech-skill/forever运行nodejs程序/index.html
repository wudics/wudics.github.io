通过forever来管理nodejs程序，对nodejs的运行进行检测，常用的就是后台运行nodejs程序和nodejs程序崩溃重启。

什么是forever

A simple CLI tool for ensuring that a given script runs continuously (i.e. forever).
即一个用来持续（或者说永远）运行一个给定脚本的简单的命令行工具。

forever可以帮助我们更好的管理Node服务，原理就是在forever进程之下，创建一个Node程序的子进程。它还有一点就是在我们每次修改程序源代码时，它都可以自动重启服务而不需要手动重启。
